Horse Racing System

Pair Project • JavaFX + MySQL

A database-driven horse racing management system with an interactive JavaFX UI. It supports race operations, ownership management, trainer/stable assignment, and performance analytics.

Key features

  • Add races with validation (track existence check before insertion).
  • Delete owner with cascading handling and archiving deleted horse data to old_info.
  • Move horses between stables with integrity checks.
  • Browse horses by owner last name and view related trainer details.
  • Winners + trainer/track statistics (wins, total winnings, race participation).

Tech: Java, JavaFX, MySQL, SQL (CRUD, joins, constraints)

My contribution: Database design, SQL queries, JavaFX–MySQL integration, and interface design.
