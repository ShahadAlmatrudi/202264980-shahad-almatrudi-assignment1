# Technical Documentation

## Overview
This portfolio is built as a static web application using HTML, CSS, and JavaScript, focusing on interactivity and clean UI design.

---

## Architecture

- HTML: Structure and layout
- CSS: Styling, theme system, responsiveness
- JavaScript: Logic, interactivity, API integration

---

## Key Components

### 1. Theme System
- Uses localStorage to save user preference
- Toggles between light and dark mode

### 2. Project Filtering & Sorting
- Filters projects based on input keywords
- Sorting implemented using JavaScript array methods

### 3. GitHub API Integration
- Fetches repositories dynamically
- Displays them as project cards

### 4. Contact Form
- Validates user inputs
- Displays success/error messages

---

## Performance Considerations

- Lazy loading images
- Efficient DOM updates
- Minimal external dependencies

---

## Accessibility

- ARIA attributes used
- Semantic HTML structure
- Keyboard-friendly navigation
