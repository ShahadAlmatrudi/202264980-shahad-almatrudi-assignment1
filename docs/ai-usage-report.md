# AI Usage Report

## Purpose
AI was used to assist in improving UI design, structuring code, debugging issues, and enhancing user experience.

## Key Uses
- Refining HTML structure for better accessibility
- Improving CSS design consistency and responsiveness
- Debugging GitHub Pages deployment issues
- Enhancing JavaScript logic for filtering and API integration
- Generating documentation and README structure

## Reflection
AI significantly improved development speed and helped identify issues quickly, especially during deployment and styling fixes. However, all outputs were reviewed, modified, and tested manually.

## Conclusion
AI was used as a support tool, not a replacement, ensuring full understanding and control over the final implementation.
